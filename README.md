# Lotto-Program
## 개요
자동/수동 로또 머신 프로그램을 구현하였습니다.

## 기능구현
- [x] 자동로또, 수동로또 선택가능
- [x] 오름차순, 중복 해결
- [x] 예외 처리
- [x] 1~45의 숫자 중 6개 선택하여 로또 번호 생성
- [x] 당첨시 5등까지, 낙첨 있음
